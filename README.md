# more clean

## サイト概要
掃除に関する情報をユーザーが共有できるサイト。

### サイトテーマ
一人では落とせない汚れをみんなで綺麗にしよう

### テーマを選んだ理由
掃除中に落とせない汚れが出てきた際、すぐに調べることができ試せるサイトがあれば便利だと思ったから。

### ターゲットユーザ
掃除頻度の多い主婦。

### 主な利用シーン
掃除をする中で落ちない汚れがあった際に調べてもらう。
